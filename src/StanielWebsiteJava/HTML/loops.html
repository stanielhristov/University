Цикли в Java

Цикълът е конструкция в програмирането, която позволява на програмата да изпълни определен блок от код повече от веднъж. Цикълът се използва, когато е необходимо да се повторят определени действия многократно, докато се изпълни определено условие.

For цикъл

For цикълът се използва, когато е нужно да се изпълни определен блок от код точно определен брой пъти. За целта се задават начална стойност, крайна стойност и стъпка за промяна на индекса.

Пример:

		for (int i = 1; i <= 5; i++) {
			System.out.println("Стъпка " + i);
		}
  • int i = 1; - задава началната стойност на променливата, която се използва за цикъла.

    • While цикъл

    While цикълът се използва, когато броят на итерациите не е известен предварително и зависи от изпълнението на определено условие. Докато условието е изпълнено, блокът от код в тялото на цикъла се изпълнява повторно и повторно.

    Пример:

    	int i = 1;
	while (i <= 5) {
		System.out.println("Стъпка " + i);
		i++;
	}
    • int i = 1; - задава началната стойност на променливата, която се използва за цикъла.
    • while (i <= 5) { - задава условието за изпълнение на цикъла, т.е. докато стойността на променливата i е по-малка или равна на 5, блокът от код в тялото на цикъла се изпълнява.
    • System.out.println("Стъпка " + i); - този блок от код се изпълнява за всяка стъпка на цикъла и извежда на екрана поредния номер на стъпката.
    • i++; - задава стъпката за промяна на индекса на цикъла, т.е. след всяка итерация на цикъла стойността на променливата се увеличава с 1.

    Foreach цикъл

    Foreach цикълът е специален вид цикъл в Java, който се използва за обхождане на елементите на масиви и колекции. Всяка итерация на цикъла взема поредния елемент от масива или колекцията и го обработва, като не е необходимо да се използва индекс.

    Пример:

    		String[] names = {"Станиел", "Димитър", "Дария", "Анна"};
		for (String name : names) {
			System.out.println("Здравей, " + name + "!");
		}
    • String[] names = {"Станиел", "Димитър", "Дария", "Анна"}; - декларира и инициализира масив от тип String с четири елемента.
    • for (String name : names) { - задава foreach цикъл, който обхожда елементите на масива и за всяка итерация взема поредния елемент и го записва в променливата name.
    • System.out.println("Здравей, " + name + "!"); - този блок от код се изпълнява за всяка итерация на цикъла и извежда на екрана поздравително съобщение за всяко име в масива.
